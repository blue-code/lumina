Lumina API Documentation

RESTful API endpoints for managing requests, executing HTTP calls, and viewing history

Project Management

GET /api/project
Get current project information
Response: 
{
  "name": "Sample API Project",
  "folder": {
    "id": "...",
    "name": "Root",
    "requests": [...]
  }
}
POST /api/project/save
Save project to file
Request Body: 
{
  "file_path": "my_project.json"
}
POST /api/project/load
Load project from file
Request Body: 
{
  "file_path": "my_project.json"
}

Request Management

GET /api/requests
Get all requests
Response: 
[
  {
    "id": "...",
    "name": "Get Users",
    "method": "GET",
    "url": "https://api.example.com/users",
    "headers": {...},
    "params": {...}
  }
]
GET /api/requests/{request_id}
Get specific request by ID
POST /api/requests
Create new request
Request Body: 
{
  "name": "New Request",
  "url": "https://api.example.com/endpoint",
  "method": "GET"
}
PUT /api/requests/{request_id}
Update request
Request Body: 
{
  "name": "Updated Name",
  "url": "https://api.example.com/new-endpoint",
  "method": "POST",
  "headers": {"Content-Type": "application/json"},
  "params": {"key": "value"},
  "body_raw": "{\"data\": \"value\"}",
  "auth_type": "bearer",
  "auth_bearer_token": "your-token-here"
}
DELETE /api/requests/{request_id}
Delete request
POST /api/requests/{request_id}/execute
Execute HTTP request and save to history
Response: 
{
  "status_code": 200,
  "status_text": "OK",
  "headers": {...},
  "body": "...",
  "elapsed_ms": 123.45,
  "size_bytes": 456,
  "content_type": "application/json"
}

History

GET /api/history/{request_id}
Get history for specific request
Query Parameters:
Parameter Type Description
limit integer Maximum number of history entries (default: 20)
GET /api/history
Get all history entries
DELETE /api/history/{request_id}
Clear history for specific request
DELETE /api/history
Clear all history

Import/Export

POST /api/import/markdown
Import API collection from markdown
Request Body: 
{
  "content": "# API Collection\n\n## Request Name\n..."
}
GET /api/export/markdown
Export all requests as markdown
Response: 
{
  "success": true,
  "content": "# Project Name\n\n## Request...",
  "request_count": 5
}

Authentication Types

Lumina supports multiple authentication methods:

No Auth 
{
  "auth_type": "none"
}
Basic Auth 
{
  "auth_type": "basic",
  "auth_basic_username": "user",
  "auth_basic_password": "password"
}
Bearer Token 
{
  "auth_type": "bearer",
  "auth_bearer_token": "your-jwt-token-here"
}
API Key 
{
  "auth_type": "apikey",
  "auth_api_key_name": "X-API-Key",
  "auth_api_key_value": "your-api-key",
  "auth_api_key_location": "header"  // or "query"
}

Environment Variables

GET /api/environments
Get all environments
GET /api/environments/active
Get active environment
POST /api/environments/active
Set active environment
Request Body: 
{
  "environment_id": "environment-uuid-here"
}
